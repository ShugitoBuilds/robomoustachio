When AI agents transact with each other on the blockchain, they have no way to know if the other agent is trustworthy. Robomoustachio solves this by computing trust scores from on-chain reputation data ‚Äî so agents can verify before they send funds.

Think of it as a credit score for AI agents. We read public feedback from the ERC-8004 Reputation Registry, run it through our scoring algorithm, and serve an instant trust verdict.

üîó Integration Example

Add trust verification to your agent in a few lines:

JavaScript 
// Before transacting with another agent
async function shouldTransact(agentId) {
  const res = await fetch(
    `https://robomoustach.io/score/${agentId}`
  );
  const { score, confidence } = await res.json();

  if (score >= 700 && confidence > 0.5) {
    return true;  // Trusted ‚Äî proceed
  }
  if (score >= 400) {
    return 'verify'; // Caution ‚Äî extra checks
  }
  return false; // Risky ‚Äî abort
}
